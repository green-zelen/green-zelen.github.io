UI/UX Design by Mariia Perlovskaia

Hi! My name is Mariia and I'm in love with creating interfaces for b2b/b2c.

That's my CV.

Tomoru / Web / 2020

TOMORU is the graphic constructor for audio and text bots, and the set of admin panels serving ready-made bots.

Goals: to visualize algorithms of bots with complex business logic, work with intents and entities, move most of the process of building bots from code to constructor app, create a design system for all business projects (main constructor, administrative panels).

Unibid / Web / 2016-2019

Unibid is a consolidated procurement management service for retail chains from the CIS countries, designed to optimize the cooperation of retail chains with manufacturers from China through the South China Service Group.

Goals: create a unified system for purchasing wholesale goods with a complete document flow of transactions, visualization of logistics instead of the classic conduct of transactions via calls/email.

Viamme / Web / 2019

Viamme is a small startup for booking excursions and selling text-based travel guides from locals/tour guides. The project is now frozen.

Goals: create the first version of the site with the minimum functionality for choosing excursions and searching for text guides, with the ability for guides to offer their services on the site through their personal account and control reservations.

Voipscan / New features / iOS / 2018

Add-on to the main Voipscan application: online recordings via chat with automatic text/audio message decryption and adding appointments on the business calendar.

Goals: to add a tool for decrypting entries and integration with the calendar.

Voipscan / Web / 2018

Simple web-version for Voipscan, without cool features of the main app.

Voipscan / iOS / 2017

Voipscan is a messenger for businesses that allows you to contact the organizations available in the database by category to order services and goods, as well as the app for budget foreign calls.

Goals: create a typical interface for text chats and calls with the ability to communicate not only with a contact list but also with organizations from the database, divided by categories of services and goods.

Jivu Sport / iOS / 2017

Goals: create an application with a feed of sports events, which can be signed up online. To implement recommendations for the user on location and opportunity to purchase tickets for events. To add a news aggregator with recommendations for interests and subscriptions for specific sports types.

ServiceMall / Web / 2017

ServiceMall is a website for hiring people for one-off services (such as renovations, cleaning, etc.)

Goals: create the first version of the interface for starting the project in one of the regions of Russia. To develop a search and selection of specialists, recommendations, a rating system, and ordering.

Habar / Android / 2015-2016

Habar is a messenger for ethnic Muslims of the CIS, taking into account cultural and religious characteristics and translating local languages within chats.

Goals: develop a chat with the ability to translate into most languages, add features for the cult, taking into account the peculiarities of the religion.

Tomoru / Web / 2020

Tomoru – это графический конструктор аудио и текстовых ботов, а так же комплекс админок, обслуживающих готовых ботов.

Задача: визуализировать алгоритмы ботов со сложной бизнес-логикой, работу с интентами и сущностями, вынести большую часть процесса сборки ботов из кода в браузер, создать дизайн-систему для всех проектов бизнеса (конструктор, административные панели).

Unibid / Web / 2016-2019

Unibid – это сервис для управления консолидированными закупками для сетевого ритейла из стран СНГ, созданный чтобы оптимизировать сотрудничество торговых сетей с производителями из Китая через посредника South China Service Group.

Задача: создать единую систему закупа оптовыми товарами с полным документооборотом сделок, визуализацией логистики взамен традиционного ведения сделок через звонки/электронную почту.

Viamme / Web / 2019

Viamme – это небольшой стартап для бронирования экскурсий и продажи текстовых туристических гайдов от локальных жителей/экскурсоводов. Сейчас проект заморожен.

Задача: Сделать первую версию сайта с минимальным функционалом выбора экскурсий и поиска текстовых гайдов, с возможностью для отдельных гидов предлагать свои услуги самостоятельно на сайте через личный кабинет и контролировать брони.

Voipscan / Дополнительные фичи / iOS / 2018

Дополнение к основному приложению Voipscan: онлайн-записи через чат с автоматической расшифровкой текста/аудио сообщения и занесением в календарь бизнес-аккаунта.

Задача: добавить инструмент для расшифровки записей и интеграцию с календарем.

Voipscan / Web / 2018

Минимальный перенос интерфейса чатов Voipscan из мобильного приложения на десктоп.

Voipscan / iOS / 2017

Voipscan – это мессенджер для бизнеса, который позволяет связываться с доступными в базе организациями по категориям для заказа услуг/товаров, а так же это бюджетная заграничная звонилка.

Задача: создать типичный интерфейс для текстовых чатов и звонков с возможностью связываться не только с контакт-листом, а еще и с организациями из базы, разделенными по категориям услуг/товаров.

Jivu Sport / iOS / 2017

Задача: сделать приложение с лентой спортивных событий, на которые можно записаться онлайн, реализовать рекомендации по локации пользователя и покупку билетов на мероприятия. Добавить агрегатор новостей с рекомендациями по интересам и подпискам на конкретные виды спорта.

ServiceMall / Web / 2017

ServiceMall - это веб-сайт для найма людей для выполнения любых услуг разной степени профессионализма (таких как ремонт, уборка и т.д.)

Задача: реализовать первую версию интерфейса для запуска проекта в одном из регионов России. Разработать поиск и подбор специалистов, рекомендации, рейтинговую систему и оформление заказов.

Habar / Android / 2015-2016

Habar – это мессенджер для этнических мусульман СНГ, с учетом культурных и религиозных особенностей и переводом локальных языков внутри чатов.

Задача: разработать чат с возможностью перевода на большинство языков, добавить религиозные фичи с учетом особенностей культа.

Насколько мне известно, приложение до сих пор работает, но они сменили дизайн в 2017 году.
